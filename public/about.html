About OpenShift Dedicated Demo 1.0.0

This application is a simple Node.js application that is expected to run in a container deployed by OpenShift. It was created to help explore OpenShift Dedicated by allowing the user to write messages to the log, crash the application, and even create and view files in a shared file system.

This project build's on a similar project by IBM, KubeToy.

This app uses Bootstrap 4 and some custom styling. All the files have been copied locally to this application so Internet connectivity is not required and can work in an air-gapped system.

The app was written by Jim Conallen <jconallen@us.ibm.com> and updated with custom OpenShift functionality by Will Gordon <wgordon@redhat.com>. The app is made available under the Apache 2.0 license. If you find any bugs, create an issue on the GitHub site, and we'll do what we can to address them.
